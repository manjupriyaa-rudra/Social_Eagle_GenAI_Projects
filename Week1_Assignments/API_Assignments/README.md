# Social Eagle Motivation Projects

# Notes: Refer to the images section for the output results

# 1. Flask Personalized Quote Generator

Objective: Provide daily motivational quotes to users based on selected categories.
Frameworks Used: Flask, Python datetime, random.
Output Example:
Request: /quote?name=Rudra&type=power
Response: 
{
  "team": "Social Eagle Batch 5",
  "category": "Power Quote",
  "quote": "Rudra, keep leading with discipline and consistency."
}

# 2. Streamlit Morning Motivation App

Objective: Display daily motivational quotes over images with a personalized greeting.
Frameworks Used: Streamlit, PIL (Python Imaging Library), random, requests.
Output Example:

Login with your name → “Good Morning, Rudra”

Image displayed with quote:
“Hard work beats talent when talent doesn’t work hard.”
