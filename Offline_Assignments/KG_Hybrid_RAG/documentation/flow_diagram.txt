=========================================
1. Overall Code Flow (High-Level Diagram)
=========================================

User launches Streamlit app
        │
        ▼
app.py (UI + controller)
        │
        ├── Upload diagram
        │
        ▼
utilities/vision_service.py
(Image → Text extraction)
        │
        ▼
pipeline/knowledge_graph.py
(Ingest text into Knowledge Graph)
        │
        ▼
pipeline/hybrid_retriever.py
(Build hybrid search index)
        │
        ▼
User asks question
        │
        ▼
utilities/chat_service.py
        │
        ├── KG retrieval
        ├── Hybrid retrieval
        ├── Context merge
        ├── LLM answer
        │
        ▼
utilities/result_logger.py
(Log results to JSON)
        │
        ▼
Answer returned to app.py
        │
        ▼
Displayed in Streamlit UI
