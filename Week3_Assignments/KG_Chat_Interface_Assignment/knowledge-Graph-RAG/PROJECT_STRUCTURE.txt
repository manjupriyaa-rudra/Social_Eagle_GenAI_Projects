APPLICATION FLOW
================================================================

1. Upload diagram via Streamlit
2. Convert image to architecture text
3. Build Knowledge Graph in Neo4j
4. Ask questions in chat
5. Get KG-grounded answers


RUNNING THE SYSTEM
================================================================

1. Start Neo4j
   docker-compose up -d

2. Activate environment
   kgenv\Scripts\activate

3. Run Streamlit
   streamlit run streamlit_app.py
